# 🔔 도전! 골든벨 (Let's Go Golden Bell)

부서 생일 이벤트용 **실시간 골든벨 퀴즈 게임**입니다. 진행자가 큰 화면에 문제를 띄우고,
참가자들은 각자 휴대폰으로 접속해 답을 제출합니다. 틀리면 탈락하는 전통 골든벨 방식이며,
최후까지 살아남으면 "골든벨"을 울립니다.

사내 로컬 PC 한 대에서 서버를 실행하면, 같은 네트워크(와이파이/사내망)에 있는
PC와 휴대폰에서 접속할 수 있습니다. 별도 클라우드/계정 설정이 필요 없습니다.

## 실행 방법

1. [Node.js](https://nodejs.org) 설치 (16 이상 권장)
2. 프로젝트 폴더에서 의존성 설치:
   ```bash
   npm install
   ```
3. 서버 실행:
   ```bash
   npm start
   ```
4. 콘솔에 표시되는 주소로 접속:
   - **호스트(진행자)**: `http://localhost:3000/host.html`
   - **대형 화면(빔/TV)**: `http://localhost:3000/screen.html`
   - **참가자(휴대폰)**: `http://<이 PC의 사내 IP>:3000/`

## 사내 IP 확인법

참가자 휴대폰은 `localhost`가 아니라 **호스트 PC의 IP 주소**로 접속해야 합니다.

- **Windows**: 명령 프롬프트에서 `ipconfig` → "IPv4 주소" (예: `192.168.0.12`)
- **macOS/Linux**: 터미널에서 `ifconfig` 또는 `ip addr` → `inet` 주소

예) IP가 `192.168.0.12`라면 참가자는 휴대폰 브라우저에서
`http://192.168.0.12:3000/` 으로 접속합니다. (같은 와이파이에 연결되어 있어야 함)

## 진행 순서

1. **문제 준비**: 호스트 화면 → `문제 관리` 탭에서 객관식/주관식 문제를 추가합니다.
   (문제는 `data/questions.json`에 자동 저장되어 재실행해도 유지됩니다.)
2. **입장**: 참가자들이 휴대폰으로 접속해 이름을 입력하고 입장합니다.
   호스트 화면 `게임 진행` 탭에서 입장자가 실시간으로 보입니다.
3. **게임 시작**: `▶ 게임 시작` → 첫 문제가 출제됩니다.
4. **타이머**: `⏱ 타이머 시작`을 누르면 제한 시간이 흐릅니다. (시간이 끝나면 자동 마감)
5. **마감/공개**: `🔒 답안 마감` 후 `✅ 정답 공개·채점`을 누르면 자동 채점되어
   오답·미응답자가 탈락합니다. 주관식은 참가자 칩을 클릭해 정답/오답을 직접 정정할 수 있습니다.
6. **다음 문제**: `➡ 다음 문제`로 진행합니다.
7. **패자부활전**: 필요 시 `♻ 전원 부활`로 탈락자를 모두 살릴 수 있습니다.
8. 마지막 문제까지 살아남은 사람이 **골든벨** 우승자입니다. 🏆

## 기술 구성

- 프론트엔드: 순수 HTML/CSS/JavaScript (빌드 단계 없음)
- 실시간 통신: Node.js + Express + Socket.IO
- 데이터: `data/questions.json` 파일에 문제 저장 (게임 진행 상태는 서버 메모리)
